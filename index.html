E-Maxx Algorithms in English

The goal of this project is to translate at least some pages of the wonderful resource http://e-maxx.ru which provides descriptions of many special algorithms especially popular in field of competitive programming.

Articles translated so far

  • Balanced Ternary
  • Binary Exponentiation
  • Euler's Totient Function
  • Suffix Array
  • Z-function
  • Linear Diophantine equation
  • Module Inverse
